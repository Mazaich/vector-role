# Vector Role

Устанавливает и настраивает Vector (лог-процессор) на CentOS/RHEL.

## Переменные роли 

Все переменные определены в файле [`defaults/main.yml`](defaults/main.yml):

| Переменная | Описание | Значение по умолчанию |
|------------|----------|------------------------|
| `vector_version` | Версия Vector для установки | `0.34.0` |
| `vector_install_dir` | Директория установки | `/opt/vector` |
| `vector_config_dir` | Директория для файлов конфигурации | `/etc/vector` |
| `vector_data_dir` | Директория для данных Vector | `/var/lib/vector` |


## Зависимости

Нет.

## Что делает эта роль

- Скачивает архив Vector с официального сайта
- Распаковывает его в `/opt/vector`
- Создаёт символическую ссылку `/usr/local/bin/vector`
- Предоставляет шаблон для конфигурации Vector (при необходимости)

## Требования

- Ansible версии 2.9 или выше
- Поддерживаемые ОС (проверено с помощью Molecule):
  - Oracle Linux 8
  - Ubuntu 20.04/22.04 (latest)

## Пример плейбука

```yaml
- name: Установка Vector
  hosts: all
  roles:
    - vector-role
  vars:
    vector_version: "0.34.0"

## Теги
- `vector` – все задачи, связанные с установкой Vector

## Тестирование с помощью Molecule
Для автоматизированного тестирования роли используется Molecule. Доступны два сценария:

### Сценарий по умолчанию (Docker)
Тестирует роль на Oracle Linux 8 и Ubuntu latest с использованием драйвера Docker.

```bash
molecule test -s default

### Сценарий Podman (облегчённый)
Тестирует роль на Ubuntu с использованием драйвера Podman (быстрее, одна платформа).

```bash
molecule test -s podman

## Запуск тестов с помощью Tox
Tox автоматизирует запуск Molecule-тестов в разных окружениях Python и Ansible. Это гарантирует совместимость роли с различными версиями.

```bash
tox

Тестируются следующие комбинации:
- Python 3.7 + Ansible 2.9
- Python 3.7 + Ansible 3.0
- Python 3.9 + Ansible 2.9
- Python 3.9 + Ansible 3.0

## Идемпотентность
Роль является идемпотентной – повторный запуск не вносит изменений в систему после первого успешного применения. Это проверяется на этапе `idempotence` в Molecule.
